# PopX UI Implementation

This is a React.js application implementing the PopX UI assignment design. It was built following strict requirements for pixel-perfect layout and responsive behavior using a centered mobile frame container.

## Tech Stack
- **React 18** (scaffolded via Vite)
- **React Router v6**
- **Vanilla CSS**

## File Structure
The project is organized efficiently with zero dead code or unused files:

- `src/components/`: Reusable components (`Button.jsx`, `Input.jsx`)
- `src/pages/`: The 4 main application screens (`Landing`, `SignUp`, `Login`, `Account`)
- `src/App.jsx`: Main application layout, state, and routing
- `src/index.css`: Global styles, layout styling, and CSS variables

## Running the Project

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the browser at the provided localhost link (typically `http://localhost:5173`).
